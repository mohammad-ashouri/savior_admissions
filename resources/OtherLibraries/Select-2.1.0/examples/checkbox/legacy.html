Select example Legacy (v1 compatibility)

Prior to Select 2 checkbox selection was possible, but it used CSS to create the visual representation of a checkbox, rather than a real input element, which is now possible in Select 2.

For backwards compatibility the old checkbox styling is still possible in Select 2, as shown in this example. It is done by assigning the class select-checkbox to the target column.

You are strongly encouraged to use the new checkbox method in new projects as it improves keyboard accessability.
